PROJECTS

SEARCHDOC

SEARCHDOC is a responsive web app, It is aimed at providing user options to search and find healthcare providers in their area. This app solves the common problem of identifying providers participating in their insurance network. SEARCHDOC is a CRUD app leveraging Firebase as its database and Uses a BetterDoctor API to get the relevant data. This app is build on the top of AngularJs and Materialize.


I developed this app for the residents and immigrants in different parts of US. Now I have few people who is using my app in real time. I appreciate Better Doctor team, for the API which holds all the critical data about the providers.
